Panduan Singkat Web Setting

Bagian ini dipakai untuk mengubah informasi website tanpa menyentuh file kode.

  • Key: nama pengaturan unik, contoh office_address atau ig_url.
  • Group: kelompok setting agar mudah dilacak, contoh site, office, ppid, social.
  • Tampil untuk publik: aktifkan jika data boleh dibaca halaman website.
  • Value: isi nilai pengaturan, bisa teks, angka, atau tautan.
  • Alur cepat: pilih setting di tabel kanan, ubah di form kiri, lalu klik Simpan Setting.
